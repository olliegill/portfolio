  • Upstage

    Upstage was my final project in the Iron Yard Academy. I teamed up with a fellow student to build a customized, white-label, micro-social network and scheduling application for music festivals and business conferences. After both of us made the data model, I concentrated on the design and making sure it was fully modular and responsive, as well as being responsible for building the schedule. This app is built using Ember and Ember-data with Firebase as a back-end. Liquid fire was used for transition animation. It was designed mobile-first and uses the Bourbon Neat framework for its responsive design.

  • Tandem

    Tandem is an app that provides a social network for people wanting to find others who speak the same language anywhere in the world, or find help learning a language on a one-to-one basis. As a team, we built the whole app and business model in 50 hours as part of Start-up Weekend Greenville. It is built with Backbone and uses Parse as a backend, which I had to learn in a day. I handled a lot of functionality, UX, and worked with a design team to make a clean, easy-to-use app.

  • Swamp Rabbit Cycling

    Swamp Rabbit Cycling was the first group project I was involved with and the first time collaborating through Github. It was built mobile-first using Bourbon and Neat and was designed from two wireframes, mobile and desktop. We used Flickr's API for the images and Googlemap's API for the maps. We also used Hammerjs for the swipe functionality on mobile. Aside from the functionality I was responsible for most of the UX/UI. We also used Bitters to keep the styles clean and segmented.

  • Super Street Fightin' Mario Brothers

    Super Street Fightin' Brothers is a turn-based game I created with two others. It was built using JQuery and Underscore and was my introduction to using prototypes and constructors, as well as test-driven development. We all created a good and bad guy each and randomized their strike-rates and attack strength. I was also responsible for the bulk of the design and for creating an old-school gaming feel.
